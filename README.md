# **Animais Fantásticos**

### 📖 **Sobre o Projeto**
Este projeto foi desenvolvido durante o curso de JavaScript ES6+ da Origamid. O objetivo foi criar uma página interativa que explora funcionalidades avançadas do JavaScript moderno, utilizando técnicas de manipulação do DOM, eventos e animações para melhorar a experiência do usuário.

O site apresenta informações sobre animais, com funcionalidades como navegação por abas, animações suaves, e efeitos interativos.

### 🛠️ **Tecnologias Utilizadas**
- **HTML5**  
- **CSS3**  
- **JavaScript (ES6+)**

### 📦 **Funcionalidades**
- Navegação por abas para exibir informações de forma organizada.  
- Animações suaves utilizando CSS e JavaScript.  
- Scroll suave para melhorar a navegação no site.  
- Manipulação do DOM para criar uma experiência dinâmica e interativa.
